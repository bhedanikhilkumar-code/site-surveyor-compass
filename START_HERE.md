## ✅ WHAT HAPPENS AUTOMATICALLY

### Phase 1: Setup (First Time)
```
SETUP_FLUTTER.ps1 does:
1. Checks if Flutter already installed
2. If not, downloads Flutter SDK
3. Extracts to C:\flutter
4. Sets FLUTTER_HOME environment variable
5. Adds to system PATH
6. Runs flutter doctor
7. Accepts Android licenses
8. Verifies everything works
```

### Phase 2: Build
```
BUILD_APK.bat does:
1. Checks Flutter installation
2. Checks Android SDK installation
3. Runs: flutter pub get
4. Runs: flutter build apk --release
5. Shows APK location and size
6. Shows next steps
```

### Phase 3: Install
```
BUILD_AND_INSTALL.bat does:
1. Cleans previous builds
2. Gets latest dependencies
3. Builds APK
4. Waits for Android device
5. Runs: flutter install --release
6. Launches app on phone
7. Shows success/error messages
```

## 📱 REQUIREMENTS

### Your Computer
- ✅ Windows 10 or 11
- ✅ 2+ GB free disk space
- ✅ Internet connection (downloads ~1 GB)
- ✅ Administrator access (for installations)

### Your Phone
- ✅ Android 5.0 or higher
- ✅ USB cable
- ✅ USB debugging enabled
- ✅ ~50 MB free space

### Enable USB Debugging on Phone
```
1. Settings → About Phone
2. Find "Build Number"
3. Tap it 7 times
4. Developer Options appears in Settings
5. Enable "USB Debugging"
6. Approve "Allow USB Debugging" when connecting
```

## 🆘 TROUBLESHOOTING

### "I don't see a menu when I run QUICK_START.bat"
→ Make sure you're running as Administrator
→ Right-click → Run as Administrator

### "Flutter download failed"
→ Check your internet connection
→ Download Flutter manually: https://flutter.dev
→ Extract to C:\flutter
→ Script will detect it

### "Phone not showing up"
→ Enable USB Debugging on phone
→ Use good quality USB cable
→ Check Settings → Developer Options

### "Build still running after 30 minutes"
→ Ctrl+C to stop
→ Run QUICK_START.bat → Option 4 (Clean)
→ Try building again

### "App won't install on phone"
→ Allow "Unknown Sources" in Security settings
→ Uninstall old version first
→ Try with USB cable properly connected

## 💡 PRO TIPS

1. **First build takes longest** (15+ min) due to Gradle setup
   → Subsequent builds: 10-12 min

2. **Keep USB cable connected** while building
   → Disconnecting pauses the build

3. **Don't close window** during build
   → Build will stop and restart from scratch

4. **Reboot if issues occur**
   → Fixes environment variable issues

5. **Use WiFi for downloads**
   → Faster than mobile hotspot
